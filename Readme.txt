This project requires no special installation steps. 
You can clone https://github.com/gitapriyam/frontend-nanodegree-arcade-game
github repository to a specific folder.
if possible 
	1) Assign this folder as a virtual directory for a web server html application.
	2) Start the game pointing to index.html with the above web application
Otherwise
	1) You can start the game by selecting index.html with the web browser.
	
When the game gets over, the user will be redirected to gameover.html
The user can choose to restart the game by selecting the hyperlink provided with gameover.html. This page also provides relevant game statistics such as the maximum treasures held and the duration of the game.
